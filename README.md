# EasyBibThis is a repo for managing machine readable artifacts for the [EasyBib APIs](http://www.easybib.com). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [EasyBib APIs](http://www.easybib.com).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://www.easybib.com)- [Developer](http://www.easybib.com)- [Twitter](https://twitter.com/#!/EasyBib)- [GitHub](https://github.com/easybib)The [APIs.json](https://github.com/api-evangelist/easybib/blob/master/apis.json) index provides links to all the relevant links for the EasyBib APIs, including the OpenAPI and Postman if they are present.
